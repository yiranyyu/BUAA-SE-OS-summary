这是一份属于北航软院的 OS 复习提纲，希望能够给大家以原力！

注：本项目并**不是**为了提供关于操作系统这门课程的一个大而全的内容概括，仅仅**是**为了方便同学们学习操作系统课程而提供的一份力图简介凝练的内容概要指南。

### 中文排版规范

基本规范：

* 中英文之间需要增加空格
* 中文与数字之间需要增加空格
* 数字与单位之间需要增加空格

[详细规范点这里](https://github.com/sparanoid/chinese-copywriting-guidelines)
